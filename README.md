# 📊 Análisis Mundial 2018 vs 2022

Este proyecto analiza el rendimiento de la selección Argentina en los mundiales de Rusia 2018 y Qatar 2022 utilizando Python y Pandas.

Objetivo
Comparar el rendimiento ofensivo y defensivo entre ambos torneos.

Análisis realizado
- Limpieza de datos
- Cálculo de goles a favor y en contra
- Comparación entre torneos
- Visualización de resultados

Conclusiones
- En 2018: bajo rendimiento ofensivo y defensivo
- En 2022: mejora significativa en ataque (15 goles)
- Defensa más sólida (solo 8 goles en contra)
- Equipo más equilibrado → campeón del mundo 🏆

Tecnologías
- Python
- Pandas
- Matplotlib
